nano — Official CouchDB client for connecting to and interacting with CouchDB databases.
bcryptjs — Pure JavaScript implementation of bcrypt for securely hashing and verifying passwords without native dependencies.
class-validator — Decorator-based validation library for validating DTOs and request payloads in NestJS.
class-transformer — Utility for transforming plain JavaScript objects into TypeScript class instances and vice versa.
@nestjs/config — Official NestJS module for centralized configuration management using environment variables and configuration files.
